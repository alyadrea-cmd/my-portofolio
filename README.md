## 📄 DOKUMENTASI PENGEMBANGAN WEBSITE

### A. Deskripsi Website
Website ini merupakan platform portofolio personal interaktif berbasis *single-page application* yang dirancang menggunakan komponen **HTML5, Tailwind CSS, dan JavaScript**. Website ini berfungsi sebagai resume digital dinamis untuk merepresentasikan identitas profesional, pencapaian kompetensi nasional di bidang informatika, serta riwayat *framework* kerja sebagai *Product Manager* dan *Developer*. 

Website ini dilengkapi dengan fitur interaktif berupa galeri kegiatan berbasis *carousel slider* dan sistem *smart-fallback redirection* untuk optimasi gerbang komunikasi *email client* (Gmail Web Integration).

### B. Target Pengguna
1. **Recruiter / HR Department:** Membantu praktisi HR dalam meninjau *skillset*, linimasa perjalanan karier, serta portofolio produk nyata secara cepat dan responsif.
2. **Potensial Kolaborator & Klien:** Memfasilitasi rekan tim, *founder* startup, atau klien yang membutuhkan manajer produk dengan kapabilitas logika pemrograman yang kuat untuk berkolaborasi.
3. **Akademisi / Dosen Penguji:** Sebagai bukti implementasi praktis dari pemahaman arsitektur *front-end* dan manajemen produk teknologi.

### C. Penjelasan Desain UI/UX
* **Desain Visual & Tema:** Menggunakan pendekatan **Matte Dark Mode** (Neutral 900 & Zinc 100) untuk memberikan kesan modern, tech-focused, dan mengurangi kelelahan mata (*eye strain*). Aksen warna **Rose/Pink 500** dipilih secara komplementer untuk menciptakan *focal point* (pusat perhatian) pada elemen-elemen penting dan tombol aksi (CTA).
* **Tipografi:** Menggunakan *font family* **Inter** yang memiliki tingkat keterbacaan (*readability*) sangat tinggi pada layar digital, mendukung struktur hierarki teks dari H1 hingga paragraf secara konsisten.
* **Pengalaman Pengguna (UX Features):**
  * *Sticky Transparent Navbar dengan Scroll-Spy:* Menu navigasi tetap berada di atas dan otomatis mendeteksi posisi *scroll* pengguna (memberikan garis indikator aktif) untuk mempermudah perpindahan seksi tanpa *reload*.
  * *Interactive Gallery:* Mengubah tumpukan gambar statis menjadi *carousel components* yang bisa diklik (`Next`/`Prev`) lengkap dengan *dots indicator*, sehingga pengguna bisa mengeksplorasi dokumentasi aktivitas secara mandiri.
  * *Responsive Grid Layout:* Mengimplementasikan *breakpoint grid* Tailwind CSS (`sm:`, `md:`, `lg:`) untuk menjamin tampilan web tetap presisi, estetik, dan proporsional baik saat diakses lewat HP, tablet, maupun layar laptop/desktop.

## 🛠️ Tech Stack & Fitur Teknis
